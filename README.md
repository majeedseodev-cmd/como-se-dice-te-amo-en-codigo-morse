# Te Amo en Código Morse

La frase **te amo en código morse** es una forma creativa y romántica de enviar un mensaje secreto usando puntos y rayas.

## Código Morse para “Te Amo”

```text
T = -
E = .
A = .-
M = --
O = ---
```

Resultado final:

```text
. .- -- ---
```

Muchas personas usan este código romántico para compartir mensajes especiales en redes sociales, cartas o chats privados.
